# Microsoft IQ Integration (Foundry IQ Style)

Career Quest Canvas implements a **Foundry IQ-style grounded context layer**. This is a local architectural pattern inspired by the principles of Microsoft Foundry IQ.

## Implementation Details
- **GroundedContextService**: Acts as the data orchestrator, loading context from verified local sources (resumes, JDs, company reports).
- **CitationBuilder**: A specialized utility that collects and deduplicates citations for every piece of advice generated by the system.
- **Grounded Evidence**: Every strength, bright spot, and interview tip in the application is linked back to a specific source ID (e.g., `resume-01`).

## Benefits
- **Reduced Hallucinations**: By strictly using the grounded context, the system avoids generating false claims about the candidate or the company.
- **Trust & Transparency**: Candidates can see exactly why a certain recommendation was made, increasing confidence in the system's output.
- **Pluggability**: The architecture is designed to be easily swappable with real Microsoft Foundry IQ or Azure AI Search services.

## Current Demo Scope

This hackathon demo uses a local Foundry IQ-style adapter. It does not call live Microsoft Foundry IQ or Azure services. The goal is to demonstrate the core IQ pattern: grounded context, evidence snippets, citation tracking, and safer recommendations using fictional demo data.

## Future Real Integration Path

A production version could replace `GroundedContextService` with a real Microsoft Foundry IQ / Azure AI Search connector, while keeping the same service interface and citation output format.
